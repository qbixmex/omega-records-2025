<h1 align="center">Music Recording and Production Studio</h1>

## Install all dependencies

```bash
# NPM
npm install

# BUN
bun install
```

## Run Development Mode

```bash
# NPM
npm run dev

# BUN
bun dev
```

## Build the project

```bash
# NPM
npm run build

# BUN
bun run build
```

## Preview Locally

**Note: You must run build before running this command**

```bash
# NPM
npm start

# BUN
bun start
```
